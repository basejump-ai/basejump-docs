# Frequently Asked Questions

## Can Basejump read CSVs?
Yes, using the Athena integration you can upload CSVs and Basejump can read them and answer questions about them.

## Is Basejump planning on adding more agents?
Yes, any agents that support data analytics are within scope for us to add.
